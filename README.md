# RELAXATION APP

A simple web application to help you relax and unwind. Listen to soothing nature sounds like waterfalls, forests, rain, and fire while observing them for a calming 3-minute session each.

---

##  Features

- Listen to four nature sounds: **Waterfall**, **Forest**, **Rain**, and **Fire**.
- Each session lasts **3 minutes**, allowing time for relaxation and mindfulness.
- Minimalist and easy-to-use interface for stress relief.
